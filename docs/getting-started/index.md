# Getting Started

Get Vexil running in your Kubernetes cluster in under 5 minutes.

## Prerequisites

| Component | Version |
|:----------|:--------|
| Kubernetes | 1.26+ |
| Helm | 3.x |
| Docker | 20+ |
| Go | 1.25+ (only for building from source) |
| Node.js | 20+ (only for building from source) |

## What You'll Learn

1. [**Installation**](/vexil/getting-started/installation/) --- Deploy Vexil to your cluster with Helm
2. [**Authentication**](/vexil/getting-started/authentication/) --- Log in and manage users
3. [**Your First Flag**](/vexil/getting-started/first-flag/) --- Create and deliver a feature flag to your workloads
